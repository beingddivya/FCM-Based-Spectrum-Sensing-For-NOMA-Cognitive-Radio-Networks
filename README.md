# FCM-Based Spectrum Sensing for NOMA Cognitive Radio Networks

This repository contains MATLAB implementation for the paper:

"FCM Based Spectrum Sensing for NOMA Cognitive Radio Networks"

Authors:
Divya Yadav  
Saikat Majumder  
Ajay Singh Raghuvanshi

Conference:
2024 2nd World Conference on Communication & Computing (WCONF)

## Files

CR_NOMA_FCM1.m : Proposed FCM algorithm  
CR_NOMA_KMC.m : K-means clustering method  
CRNOMA_GMM.m : Gaussian mixture model  
confusion_matrix.m : Confusion matrix evaluation

## Requirements

MATLAB R2019b or later
